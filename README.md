# 🔢 Number Guessing Game (Python)  

A fun **Number Guessing Game** built with Python.  

Try to guess the secret number between 1 and 100 before you run out of attempts! 🎯  

---

## 📌 About the Project  

<details>
<summary><strong>▶ What does this project do?</strong></summary>

This project is a console-based number guessing game.

The computer:

- 🎲 Randomly selects a number between **1 and 100**  
- 🎚️ Lets the player choose a difficulty level  
- 🔁 Gives feedback after each guess  
- ❤️ Limits the number of attempts  

The player must guess the correct number before running out of attempts.

This beginner-friendly project helps practice:

- 🧠 Functions  
- 🔁 `while` loops  
- 🔀 Conditional statements  
- 🎲 Random number generation  
- 📉 Managing attempts and game logic  

</details>

---

## ✨ Features  

- 🎯 Random number generation (1–100)  
- 🎚️ Difficulty levels:
  - 🟢 Easy → 10 attempts  
  - 🔴 Hard → 5 attempts  
- 🔽 "Too low" feedback  
- 🔼 "Too high" feedback  
- 🏆 Win condition  
- 💀 Lose when attempts reach zero
